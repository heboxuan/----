全民健身打造15分钟，健身权非常好，受欢迎可是很多地方都没有达到，特别是东北地区，冬季寒冷，很不方便，很多场所对外开放有限，特别是哈尔滨市，作为省会市15分健身圈市民基本都不知道，没有见到一个地方有十五分钟健身圈儿的标识，只是民间的自发的健身活动，体育部门很少组织。
麻烦局长了，请国家督促各地，特别是东北及哈尔滨市，认真落实好全民健身，尤其是冬季的健身活动，健身场所的开放。
盼望局长为我们解决烦心事，增强体质谢谢

  • 回复

    国家体育总局局长苟仲文

    　　近年来，体育总局积极会同有关部门支持地方构建“15分钟健身圈”（指在城市社区，居民从居住地步行不超过15分钟范围内，有可供开展健步走、广场舞、球类运动等群众性健身活动的场地设施）。2006年以来，体育总局会同有关部门每年利用中央资金支持地方建设全民健身场地设施，其中全民健身路径工程和农民体育健身工程等项目所配置的健身场地设施免费供群众使用。自2014年开始，体育总局会同有关部门安排中央资金支持大型公共体育场馆免费或低收费向社会开放，补助范围包括东北地区的体育场、体育馆、游泳馆和滑冰馆等。为更好地推动公共体育场馆向社会开放，体育总局正着手研究公共体育场馆开放服务有关标准，进一步细化体育场馆开放服务项目、开放时间、收费标准等要求。
    　　关于“官方”渠道组织健身活动方面。多年来，体育总局倡导和推动全国开展新年登高活动、全国“全民健身日”活动和全国群众冬季运动推广普及活动等各类全民健身主题活动，由此带动起群众性赛事活动的广泛开展。东北地区的体育部门也积极行动起来，组织举办了丰富多彩的赛事和活动。辽宁、吉林、黑龙江除了在本省范围内举办各级各类全民健身活动之外，还举办过全国全民健身主题示范活动的主会场或特色会场活动。黑龙江连续四年承办了全国群众冬季运动推广普及活动主会场。东北地区发挥自然和气候优势，通过大力开展群众冬季运动推广普及活动，进一步打造冰雪运动品牌赛事活动，发展了形式多样、各具特色的冬季体育运动项目，引领和推动全国群众性冰雪运动广泛开展，丰富冬季运动产品供给，助力“三亿人参与冰雪运动”。第十四届全国冬季运动会首次设置冰上龙舟和雪地足球项目比赛，有力带动起人们参与冰雪运动的热情。新冠肺炎疫情期间，为丰富群众的居家生活和健身需求，体育总局印发了《关于大力推广居家科学健身方法的通知》，推广简便易行、科学有效的居家健身方法，同时，推动运动项目中心（协会）、各省（区、市）体育部门和社会力量举办丰富多采的全民健身网络赛事和活动，为广大体育爱好者足不出户参与赛事活动搭建平台、创造了条件。
    　　目前，各地在“15分钟健身圈”建设等方面还存在发展不平衡、服务水平不高等问题。我们将结合你的建议，统筹考虑区域差异、季节差异等问题，同时进一步加强对场馆开放工作的监督指导，推动东北地区体育部门加大组织健身赛事和活动，不断改进工作，推动各地进一步提升全民健身公共服务水平，更好地满足群众对健身场地设施和参与群众体育活动的需求。
    　　感谢网民朋友们对体育事业的关心和支持。
